## 🚀 Features

- 🧠 OCR (Optical Character Recognition) for scanned documents.
- 🔍 Full-text search and document filtering.
- 🏷️ Auto-tagging and categorization.
- 🌐 Web-based user interface.
- 🐳 Docker-based deployment.
